# TG WS PROXY MAGISK by F1NDLE

Высокопроизводительный Telegram MTProto WebSocket Bridge Proxy, переписанный на **Rust**.

## Структура
* **system/bin/tg-ws-proxy** - бинарник на Rust.
* **action.sh** - скрипт-тоггл для управления прокси.
* **module.prop** - информация о модуле.

## Установка и запуск
1. Установите архив через **Magisk/KSU**.
2. Перейдите в раздел установленных модулей.
3. Найдите модуль **TG WS Proxy** и нажмите кнопку **Action** (Действие).
4. Дождитесь инициализации (1-3 секунды).
5. После автоматического открытия Telegram подтвердите добавление прокси сервера.

## Управление
Нажатие кнопки **Action** работает как выключатель:
* Если прокси выключен - он запустится и откроет ссылку в приложении.
* Если прокси запущен - он будет полностью остановлен.
