# TG WS Proxy Magisk (F1NDLE Edition)

Данный модуль Magisk реализует прокси-мост MTProto через WebSockets для Telegram. Решение предназначено для обеспечения стабильного соединения в сетях с глубокой фильтрацией трафика (DPI).

## Основные функции
* Поддержка MTProto: Использование актуального протокола Telegram для безопасности данных.
* WebSocket Туннелирование: Инкапсуляция трафика в WS-потоки для обхода блокировок.
* Интеграция с Magisk Action: Управление работой модуля через интерфейс Magisk.
* Автоматический запуск Intent: При активации система автоматически открывает Telegram с готовой конфигурацией прокси.

## Технические требования

| Параметр | Требование |

| ОС Android | Версия 7.0 и выше |
| Версия Magisk | 20.4+ или актуальный KernelSU |

## Установка
1. Скачайте архив модуля.
2. Установите его через вкладку "Модули" в приложении Magisk или KernelSU.
3. Перезагрузите устройство для применения изменений.

## Использование
1. Откройте Magisk и перейдите в раздел установленных модулей.
2. Найдите модуль TG WS Proxy и нажмите кнопку Action (Действие).
3. Дождитесь завершения инициализации (около 3-5 секунд).
4. После автоматического открытия Telegram подтвердите добавление прокси-сервера в приложении.

## Техническая реализация
Модуль разворачивает локальный сервис по адресу 127.0.0.1:1443. Скрипт на Python обрабатывает входящие соединения, шифрует их с помощью библиотеки cryptography и перенаправляет на выбранные дата-центры Telegram через WebSocket-туннель, маскируя пакеты под стандартный веб-трафик.
