
# Че изменилось?
Autostart: Реализована функция автоматического запуска при загрузке системы (требует статический секрет в конфиге).

Dynamic Arguments: Добавлена поддержка опциональных секретов, Cloudflare доменов и любых дополнительных флагов через EXTRA_ARGS.

Config System: Добавлена настройка через config.conf в корне модуля

Logging: Улучшена система логирования. Все ошибки и ссылка теперь записываются в proxy.log.

Binary: Обновлен бинарник
